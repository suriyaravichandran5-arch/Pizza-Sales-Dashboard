# 🍕 Pizza Sales Dashboard

## 📊 Project Overview
This project analyzes pizza sales data to identify best-selling and worst-selling pizzas using Power BI, SQL, and Excel.

## 📈 Key Metrics
- **Total Revenue:** $817.86K
- **Total Orders:** 21,350
- **Total Pizzas Sold:** 49,574
- **Average Order Value:** $38.31

## 🏆 Top Performers
- **Highest Revenue:** Thai Chicken Pizza
- **Highest Quantity:** The Classic Deluxe Pizza
- **Highest Orders:** The Classic Deluxe Pizza

## 📉 Bottom Performers
- **Lowest Revenue & Quantity:** The Brie Carre Pizza
